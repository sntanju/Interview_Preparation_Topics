# IPv4 vs IPv6

Internet Protocol (IP) is used to identify devices on a network. IPv4 and IPv6 are the two major versions of the Internet Protocol.

---

## 🌐 IPv4

IPv4 (Internet Protocol Version 4) is the most widely used version of IP.

### Features:
- 32-bit address length
- Decimal notation
- Approximately 4.3 billion unique addresses

### Example:
192.168.1.1

---

## 🌐 IPv6

IPv6 (Internet Protocol Version 6) was introduced to overcome IPv4 address exhaustion.

### Features:
- 128-bit address length
- Hexadecimal notation
- Extremely large address space
- Better scalability and future support

### Example:
2001:db8::1

---

## ⚖️ Comparison

| Feature | IPv4 | IPv6 |
|----------|-------|-------|
| Address Length | 32-bit | 128-bit |
| Format | Decimal | Hexadecimal |
| Example | 192.168.1.1 | 2001:db8::1 |
| Address Space | ~4.3 Billion | ~340 Undecillion |
| NAT Requirement | Common | Usually Not Required |
| Adoption | Widely Used | Increasing |

---

## 🚀 Why IPv6?

IPv6 was developed to solve the limited address space problem of IPv4 and provide a scalable solution for the growing number of internet-connected devices.

---

## 🧠 Interview Answer (One Line)

IPv4 uses 32-bit addresses and supports about 4.3 billion unique addresses, while IPv6 uses 128-bit addresses and provides an extremely large address space for future internet growth.
