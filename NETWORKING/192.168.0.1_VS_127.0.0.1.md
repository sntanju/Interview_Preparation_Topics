# 192.168.0.1 vs 127.0.0.1

## 🏠 192.168.0.1 (Private IP)

192.168.0.1 is a private IP address commonly used as a default gateway for routers in local networks.

### Key Points:
- Used in LAN (Local Area Network)
- Typically assigned to routers
- Allows devices to communicate within the same network
- Used to access router configuration page

### Example:
- Router: 192.168.0.1
- Devices: 192.168.0.2, 192.168.0.3

---

## 🔁 127.0.0.1 (Loopback Address)

127.0.0.1 is a loopback IP address that points back to the same device.

### Key Points:
- Used for localhost testing
- No network required
- Refers to your own machine
- Used in development environments

### Example:
- http://127.0.0.1 → local server

---

## ⚖️ Comparison

| Feature | 192.168.0.1 | 127.0.0.1 |
|----------|--------------|------------|
| Type | Private IP | Loopback IP |
| Scope | Local network | Same device |
| Usage | Router / LAN devices | Local development |
| Internet | Not required | Not required |
| Access | Multiple devices | Only one device |

---

## 🧠 Interview Answer (One Line)

192.168.0.1 is a private network IP used for local devices and routers, while 127.0.0.1 is a loopback address used to refer to the local machine itself.
