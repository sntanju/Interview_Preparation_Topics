# Public Network vs Private Network

## 🌐 Public Network

A Public Network is a network that is openly accessible to anyone without restriction. The Internet is the largest example of a public network.

### Examples:
- Internet
- Public WiFi (cafes, airports, malls)

### Key Features:
- Open access
- Lower security
- Shared bandwidth
- Limited control

---

## 🔒 Private Network

A Private Network is a secure and restricted network that allows access only to authorized users or devices.

### Examples:
- Home WiFi network
- Office network
- Corporate intranet

### Key Features:
- Restricted access
- High security
- Better control
- Stable performance

---

## ⚖️ Comparison

| Feature | Public Network | Private Network |
|----------|----------------|----------------|
| Access | Open to all users | Restricted to authorized users |
| Security | Low | High |
| Performance | Shared and variable | Stable and optimized |
| Control | No control | Full administrative control |
| Example | Internet | Office/Home network |

---

## 🌍 Example

Using YouTube on mobile data is an example of a public network, while accessing company internal systems via office WiFi is an example of a private network.

---

## 🧠 Interview Answer (One Line)

A public network is an open network accessible to everyone, while a private network is a secure network restricted to authorized users only.
