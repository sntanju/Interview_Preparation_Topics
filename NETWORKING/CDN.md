# CDN (Content Delivery Network)

A CDN is a geographically distributed network of servers that delivers web content (images, videos, web pages, and files) to users based on their location.

Instead of serving all users from a single origin server, a CDN stores cached copies of content on multiple edge servers around the world.

---

## ⚙️ How CDN Works

1. Website content is stored on the origin server
2. CDN replicates this content to multiple edge servers
3. When a user requests content
4. CDN routes the request to the nearest edge server
5. The content is delivered faster with lower latency

---

## 📍 Edge Server

An edge server is a CDN server located geographically closer to users to reduce latency and improve performance.

---

## 🚀 Key Benefits

### 1. Faster Load Times
Content is delivered from the nearest server, reducing delay.

### 2. High Availability
If one server fails, another edge server can serve the content.

### 3. Reduced Bandwidth Cost
Origin server handles less traffic due to caching.

### 4. Improved Security
CDNs often include DDoS protection and traffic filtering.

---

## 🌍 Example

Popular platforms like YouTube, Netflix, and Facebook use CDNs to deliver content quickly to users worldwide.

---

## 🌍 One-line Interview Answer

CDN is a network of distributed servers that delivers content from the nearest location to the user, reducing latency and improving performance, reliability, and scalability.
