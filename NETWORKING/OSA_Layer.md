# OSI Model (Open Systems Interconnection Model)

The OSI Model is a conceptual framework developed by the ISO (International Organization for Standardization) to standardize communication between different computer systems over a network.

It divides the complex process of network communication into 7 distinct layers, where each layer has a specific role and communicates with the layers above and below it.

---

## 🧱 The 7 Layers of the OSI Model

### 1. Physical Layer (Layer 1)
- Deals with physical transmission of data
- Includes cables, switches, electrical signals, radio waves, and hardware components

### 2. Data Link Layer (Layer 2)
- Responsible for node-to-node data transfer
- Handles MAC addressing and error detection
- Uses protocols like Ethernet

### 3. Network Layer (Layer 3)
- Handles routing of data between different networks
- Uses IP addressing to find the destination

### 4. Transport Layer (Layer 4)
- Ensures reliable data delivery
- Controls flow, error handling, and segmentation
- Uses TCP and UDP protocols

### 5. Session Layer (Layer 5)
- Establishes, manages, and terminates communication sessions between applications

### 6. Presentation Layer (Layer 6)
- Translates, encrypts, and compresses data
- Ensures data is readable for the application layer

### 7. Application Layer (Layer 7)
- Closest to the user
- Provides network services to applications like web browsers, email, and APIs (HTTP, SMTP, FTP)

---

## 🎯 Why OSI Model is Important

- Standardizes network communication
- Helps in troubleshooting network issues easily
- Makes complex networking easier to understand
- Allows interoperability between different systems and devices
